  * ğŸŒ **Output Multi-Bahasa Praktis:**

## ğŸš€ Panduan Penggunaan

Ikuti langkah-langkah mudah berikut untuk mulai menggunakan aplikasi:

1.  **ğŸ”‘ Pengaturan API Key (Langkah Krusial\!):**
      * Di bagian atas aplikasi, cari area **"Pengaturan API Key Gemini Anda"**.
      * Klik tombol **"Dapatkan API Key"**. Ini akan membuka tab baru ke laman Google AI Studio.
      * Ikuti instruksi di Google AI Studio untuk membuat atau menyalin API Key Anda.
      * Kembali ke aplikasi Prompt Generator, **tempelkan API Key Anda** ke dalam kolom yang disediakan.
      * Klik tombol **"Simpan API Key"**. API Key Anda kini tersimpan aman di browser Anda (menggunakan `localStorage`).
      * *Catatan Penting:*
          * Tanpa API Key yang valid, fitur yang memanggil AI (peningkaan prompt, generasi gambar, terjemahan) **tidak akan berfungsi**.
          * API Key Anda disimpan secara lokal di browser Anda dan tidak dikirim ke server kami.
2.  **ğŸ¨ Pilih Mode Prompt:** Tentukan jenis output yang Anda inginkan dengan memilih salah satu dari empat mode: Gambar, Teks ke Video, Adegan Video, atau Narasi.
3.  **ğŸ’¡ Masukkan Ide Dasar (Opsional):** Tuliskan konsep atau kata kunci utama Anda. Ini akan menjadi titik awal bagi AI untuk mengembangkan prompt.
4.  **âœï¸ Lengkapi Struktur Input:** Isi detail pada formulir yang muncul sesuai mode yang dipilih. Semakin detail input Anda, semakin baik AI dapat memahami keinginan Anda. Manfaatkan opsi "Isi Sendiri..." pada dropdown jika perlu.
      * **(Khusus Mode Gambar - Karakter Konsisten):**
          * Untuk membuat karakter yang dapat digunakan berulang kali, isi detail di bagian "Definisi Karakter Konsisten".
          * Beri nama pada profil karakter Anda di field "Nama Profil untuk Simpan/Perbarui", lalu klik "Simpan Profil".
          * Untuk menggunakan profil yang sudah ada, pilih dari dropdown "Pilih Profil Tersimpan", lalu klik "Muat Profil".
5.  **âœ¨ Hasilkan & Tingkatkan Teks Prompt:** Klik tombol ini. AI Gemini akan memproses input Anda dan menghasilkan versi prompt teks yang telah disempurnakan.
6.  **ğŸ“œ Lihat Hasil Teks Prompt:**
      * Prompt yang telah disempurnakan akan ditampilkan dalam dua bahasa: Bahasa Indonesia dan Bahasa Inggris (dialog pada mode Teks ke Video tetap dalam Bahasa Indonesia).
      * Gunakan tombol "Salin ID" atau "Salin EN" di samping masing-masing area teks.
7.  **(Mode Gambar) ğŸ–¼ï¸ Generate & Unduh Gambar:**
      * Setelah teks prompt Bahasa Indonesia berhasil ditingkatkan, tombol "ğŸ–¼ï¸ Generate Gambar dengan AI" akan aktif. Klik tombol ini.
      * Gambar akan muncul di bawah. Klik tombol "ğŸ’¾ Unduh Gambar" yang muncul di bawah gambar untuk menyimpannya.

> **ğŸ’¡ Tips:** Kualitas hasil sangat bergantung pada detail input Anda dan bagaimana AI menginterpretasikannya. Jangan ragu untuk bereksperimen dengan berbagai kombinasi input dan deskripsi untuk mendapatkan hasil terbaik\!
